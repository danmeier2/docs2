## Get Started with VectorFlow locally

Clone the repository.

```
git clone https://github.com/dgarnitz/vectorflow.git
```

```
cd vectorflow
```

**_Run the setup script._** By default this will spin up an instance of VectorFlow with Qdrant with an collection name of `blabla` as vector database locally.

```
./setup.sh
```
